Intro; why hexbins (or other shapes) are better for visualizing density than volume-based circles or choropleth maps

Tableau better for maps that aren't meant for significant user interaction (zooming, hovering, etc.), b/c backend is much slower than Mapbox and will likely result in frustratingly slow performance.

1) Tableau:

Advantages Over Mapbox:
- Easier to create - Can embed more easily in report or dashboard
- Creating hexbins doesn't require other software
- Can create pseudo-maps (e.g. each state represented by single hexagon)
- User can adjust hexbin sizing through hexbin size box (bottom right)
- Existing color patterns available (as opposed to creating from scratch)
- Can utilize Mapbox map themes in background (as demonstrated in the second map below)

Advantages Over Tableau:
